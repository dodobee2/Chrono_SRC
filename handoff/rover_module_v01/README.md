# 로버 모듈 v01 — Chrono 차량 파트 공유용

작성: 호진 (2026-07-14)

어제 도희님이 올려주신 RoverSpec 형식 그대로 스펙 파일을 만들고, 그 파일을
읽어 PyChrono 차량 모델을 조립하는 코드까지 만들었습니다. 이 폴더는 그
결과를 팀에서 보기 편하게 묶은 것입니다. 코드 전체는 제 레포(rover_challenge)에
있고, 여기에는 그림과 문서, 스펙 파일만 담았습니다.

## 뭐부터 보면 되나요

용어가 낯설면 `04_용어_풀이.md`를 먼저 열어두고 나머지를 읽으시는 걸
권합니다.

1. `images/scout_vs_main.png` — 두 로버가 어떻게 생겼는지 (파랑 scout, 빨강 main)
2. `01_로버_모듈_소개.md` — 뭘 만들었고 각 파일이 뭔지
3. `02_설계_이유.md` — 왜 이런 값과 구조로 정했는지, 팀 확인이 필요한 가정 목록
4. `03_검증_결과.md` — 시뮬레이션이 물리적으로 맞게 돌아가는지 확인한 결과

## 폴더 구성

    01_로버_모듈_소개.md     산출물 하나하나 설명
    02_설계_이유.md          설계 근거와 가정 (질량 1.5~3 kg 방침 반영)
    03_검증_결과.md          검증 6종 결과 해설 (직진·경사·선회·수납 포함)
    04_용어_풀이.md          용어 사전
    images/                  렌더 이미지, 검증 그래프
    specs/                   scout_v01.yaml, main_v01.yaml (로버 스펙 파일)
    results/summary.json     검증 수치 요약
    code/                    파이썬 코드 원본 (src, scripts, tests)

궁금한 점이나 고쳐야 할 값이 보이면 편하게 말씀해 주세요.
