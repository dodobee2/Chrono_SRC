# 코드 실행 방법

연습 레포(Project_Chrono_Practice)를 빌드해둔 컴퓨터라면 그대로 돌아갑니다.
PyChrono는 연습 레포의 빌드를 재사용하고, 경로 설정은
src/chrono_env.py가 알아서 처리합니다. 빌드 위치가 기본값
(~/Documents/Praxis/Project_Chrono_Practice/chrono_build)과 다르면
환경변수 ROVER_CHRONO_BUILD_DIR로 지정해 주세요.

    conda activate chrono
    python scripts/run_rover_check.py      # 검증 26개 전부 실행 (창 없음)
    python scripts/view_rover.py           # 3D 창으로 실시간 주행
    python scripts/view_rover.py --slope 10
    python scripts/render_rover.py         # 보고용 이미지 캡처
    python -m pytest tests/ -q             # 단위 테스트 26개

결과물은 outputs/ 아래에 CSV와 PNG로 쌓입니다.

## 파일 구성

    specs/rovers/     로버 스펙 yaml (여기 숫자만 고치면 차량이 바뀝니다)
    src/              스키마, 빌더, 테스트 바닥, 로거, 환경 부트스트랩
    scripts/          검증, 플롯, 3D 뷰어, 렌더 캡처
    tests/            pytest 단위 테스트

## 다른 파트에서 가져다 쓸 때

지형 쪽 시스템에 로버를 얹는 최소 코드입니다.

    from rover_schema import load_spec
    from rover_builder import build_rover, apply_collision_defaults

    spec, cfg = load_spec("specs/rovers/scout_v01.yaml")
    apply_collision_defaults(cfg)          # 바디 만들기 전에 한 번
    rover = build_rover(system, spec, cfg) # system은 지형 쪽에서 만든 것

    rover.set_command(12.0)                # 직진 (wheel_speed: rad/s)
    rover.set_command_lr(-6.0, 6.0)        # 제자리 좌선회

    # 시뮬레이션 루프 안에서 매 스텝:
    system.DoStepDynamics(dt)
    rover.update(dt)
